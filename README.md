# BetterLetter explainer figures

Public-safe interactive teaching figures for BetterLetter feature explainers, served via GitHub Pages and embedded in internal Notion explainer pages.

These figures deliberately exclude internal-only material - that content lives in the access-controlled Notion pages alongside each embed. Figures are self-contained HTML (Atkinson Hyperlegible via Google Fonts).

Published at: https://katechan-stack.github.io/betterletter-explainer-figures/figures/<name>.html
